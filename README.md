# rpkg
Riley's Package Manager<br>
A simple package manager built in bash

## Flags
Custom flags you can pass to change the output
- `-f`, `--force-sort`: Force the downloader to re-sort the repositories
- `-p`, `--patch`: Force the downloader to check for patches
- `-q`, `--quiet`: Suppress the majority of the output
- `-I`, `--ignore`: Ignore the checksum file and download anything from a repo
